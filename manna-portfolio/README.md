# Manna Portfolio

A soft, elegant, premium portfolio featuring a custom 3D cat hero section, glassmorphism UI cards, a dedicated image section for Marie (the aristocratic cat), and a built-in music player.

## Technologies Used
- React 19 + Vite 6
- Tailwind CSS v4
- Three.js, @react-three/fiber, @react-three/drei
- Framer Motion

## Quick Start (How to Run)

To run this project locally, make sure you are in the correct directory where this `README.md` is located. 

> **Important:** The project folder is `manna-portfolio`. If you just opened your terminal in the parent folder, you need to navigate inside first.

1. **Navigate into the project directory:**
   ```bash
   cd manna-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the site:**
   Open your browser and navigate to the local URL provided in the terminal (usually `http://localhost:5173`).

## Building for Production

To create a production-ready build:
```bash
npm run build
```
This will generate a `dist` folder which can be deployed to static hosting services like Vercel, Netlify, or GitHub Pages.
